API Documentation

To integrate with this API programmatically, use the following endpoint:

Query Endpoint

POST /query

Send a JSON object with a "query" field:

{ "query": "How do I authenticate with the GC Forms API?" }

Sample Code (Python)

import requests import json url = "http://localhost:5001/query" payload = {"query": "How do I authenticate with the GC Forms API?"} headers = {"Content-Type": "application/json"} response = requests.post(url, headers=headers, data=json.dumps(payload)) data = response.json() print(data["results"])

Sample Code (JavaScript)

fetch('http://localhost:5001/query', { method: 'POST', headers: { 'Content-Type': 'application/json', }, body: JSON.stringify({ query: 'How do I authenticate with the GC Forms API?', }), }) .then(response => response.json()) .then(data => { console.log(data.results); });
